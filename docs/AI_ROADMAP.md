# AI_ROADMAP

## Phase 1: 규칙 기반 분석

- 출석률 계산
- 노쇼율 계산
- 연속 출석 계산
- 포지션별 경기 수와 승률
- 신뢰도 점수 산정

## Phase 2: 고급 분석

- 포지션 적합도
- 멤버 참여도 추세
- 시즌별 변화
- 팀 운영 리스크 탐지

## Phase 3: AI Assistant

예시 질문:

- "최근 10경기에서 CAM 성과가 가장 좋은 멤버는?"
- "이번 달 노쇼 위험이 높은 멤버는?"
- "참석률이 떨어진 멤버에게 어떤 알림을 보내면 좋을까?"

## Phase 4: AI Coach

- 추천 포메이션
- 라인업 후보
- 포지션별 대체 멤버

## Phase 5: AI Club Manager

- 참석률 트렌드 분석
- 운영자용 주간 요약
- 노쇼 예방 알림 추천
- 유료 플랜용 인사이트 리포트
