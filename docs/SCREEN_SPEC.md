# SCREEN_SPEC

MoIja는 축구/풋살 동호회 운영 SaaS이지만, 화면 문구와 정보 구조는 다른 정기 모임에도 확장될 수 있게 작성한다. 한국 사용자를 기본 대상으로 하며, 핵심 행동은 모바일에서 1~2번 안에 가능해야 한다.

## 1. 전체 화면 원칙

- 운영자는 경기 당일 필요한 행동을 먼저 봐야 한다.
- 멤버는 본인의 참석 상태, 참석률, 신뢰도를 바로 이해해야 한다.
- 용병은 팀 가입 없이 초대받은 경기 참석 응답만 쉽게 할 수 있어야 한다.
- Coach는 경기 전 라인업과 포메이션을 빠르게 만들고 공유할 수 있어야 한다.
- 통계와 분석은 숫자만 보여주지 말고 다음 행동으로 연결되어야 한다.

## 2. 로그인 및 첫 진입

첫 화면은 한국 사용자와 운영자가 바로 이해할 수 있는 한국어 운영 화면으로 구성한다.

핵심 구성:

- 좌측 또는 모바일 상단에 `모임참석 운영 플랫폼` 브랜드와 주요 메뉴를 표시한다.
- 주요 메뉴는 `대시보드`, `모임 생성`, `참석 체크`, `공지/리마인드`를 우선 노출한다.
- 상단 로그인 영역에는 `운영자 로그인` 문구와 Kakao 로그인 버튼, 일반 계정 로그인 진입점을 배치한다.
- Kakao 로그인은 한국 사용자 기본 진입 방식으로 우선 고려한다.
- 일반 회원가입은 이메일, 비밀번호, 닉네임만 입력받는다.
- Supabase 환경 변수가 설정되지 않은 MVP 상태에서는 설정 필요 문구를 표시한다.

## 3. 하단 메뉴

- 홈
- 경기
- 랭킹
- 팀
- 내 정보

전술/라인업 기능이 활성화되면 하단 메뉴에 새 항목을 추가하지 않는다. `경기` 상세는 허브로 두고, 라인업/출석/용병/기록은 경기 상세에서 진입하는 전체화면 작업 화면으로 분리한다.

## 4. 운영자 대시보드

운영자가 가장 먼저 확인해야 하는 정보:

- 오늘 경기 또는 다음 경기
- 참석 예정, 미응답, 대기, 불참, 용병, 노쇼 위험
- 응답률
- 출석 확정 필요 인원
- 리마인드가 필요한 멤버
- 용병 초대 링크 상태

주요 행동:

- 미응답자 리마인드 보내기
- 대기자 확정
- 용병 초대 코드 생성
- 현장 출석 체크 열기
- 노쇼 처리
- 경기 결과 입력

모바일에서는 `오늘 해야 할 일` 카드가 첫 화면 위쪽에 있어야 한다.

## 5. 홈

멤버 기준 핵심 정보:

- 다음 경기
- 내 참석 상태
- 내 참석률
- 내 신뢰도
- 연속 참석
- 최근 경기 기록
- 내 포지션과 최근 배치 포지션

주요 행동:

- 다음 경기 참석 신청
- 참석 상태 변경
- 출석 체크
- 내 라인업 위치 확인
- 최근 랭킹 확인

홈 화면은 실제 조작을 직접 펼치지 않는다. 운영자와 멤버 모두 다음 행동을 누르면 해당 전체화면 작업 화면으로 이동한다.

## 6. 경기 목록

목록 정보:

- 경기 일시
- 장소
- 참석/대기/불참/미응답/용병 인원
- 신청 마감 시간
- 내 참석 상태
- 출석 방식
- 라인업 공유 여부

필터:

- 예정 경기
- 오늘 경기
- 종료 경기
- 내 참석 경기
- 기록 입력 필요 경기

## 7. 경기 상세

역할:

- 경기 상태를 빠르게 파악하는 허브
- 출석, 라인업, 용병, 기록 작업 화면으로 이동하는 진입점
- 긴 목록, 내부 탭, 해시 이동 대신 route 기반 작업 화면을 사용

상세 정보:

- 경기명, 일시, 장소, 지도
- 신청 마감 시간
- 참석 방식: 수동, QR, GPS, GPS + 승인
- 참석자 목록
- 대기자 목록
- 불참자 목록
- 용병 목록
- 라인업/포메이션 상태
- 경기 결과와 개인 기록

운영자 도구:

- 참석 확정
- 노쇼 처리
- 지각 처리
- 대기자 확정
- 용병 초대
- QR 코드 보기
- GPS 체크인 승인
- 결과 입력

멤버 도구:

- 참석/불참/대기 응답
- 출석 체크
- 내 라인업 위치 확인
- 경기 후 개인 기록 확인

작업 화면:

- `/meetings/[id]/attendance`: 운영자 빠른 체크인 또는 멤버 참석 응답
- `/meetings/[id]/lineup`: 라인업/작전판
- `/meetings/[id]/guests`: 용병 초대와 상태 관리
- `/meetings/[id]/record`: 경기 결과와 개인 기록 입력

## 8. 참석 체크 화면

운영자 현장 화면:

- 참석 예정자
- 체크인 완료자
- 미도착자
- 용병
- 노쇼 후보
- 수동 확정 버튼
- GPS/QR 판정 결과

상태 표기:

- 참석 예정
- 체크인 완료
- 운영자 확정
- 지각
- 노쇼
- 불참
- 대기

핵심 원칙:

- 운영자는 한 손으로 빠르게 상태를 바꿀 수 있어야 한다.
- GPS/QR 결과가 있어도 최종 확정은 운영자가 할 수 있어야 한다.
- 상태 변경은 `attendance_events`에 이력으로 남는다.

## 9. 용병 초대 화면

운영자 기능:

- 특정 경기용 초대 코드/링크 생성
- 만료 시간 설정
- 최대 사용 횟수 설정
- 용병 표시명 확인
- 용병 참석 상태 확인
- 용병을 라인업에 배치
- 용병 노쇼 처리

용병 진입 화면:

- 경기 정보
- 장소
- 참석 여부 질문
- 표시명 입력
- 참석/불참 버튼
- 출석 체크 버튼

Guest는 팀 멤버 목록, 시즌 통계, 내부 랭킹에 접근하지 않는다.

## 10. 랭킹

표시 정보:

- 신뢰도
- 참석률
- 연속 참석
- 노쇼율
- MVP
- 골/어시스트
- 포지션별 출전 수

운영자 관점:

- 노쇼 위험 멤버
- 참석률 하락 멤버
- 꾸준히 참석하는 멤버
- 대기 전환 후 참석률이 높은 멤버

## 11. 팀

관리 정보:

- 팀 프로필
- 멤버 목록
- 권한
- 팀 초대 코드
- 시즌
- 포지션 코드
- 홈구장

멤버 관리:

- 역할 변경
- Coach 지정
- Manager 지정
- 포지션/주발 정보 확인
- 비활성 멤버 관리

## 12. 내 정보

개인 정보:

- 프로필
- 참석률
- 신뢰도
- 연속 참석
- 포지션
- 가능 포지션
- 주발
- 개인 기록
- 배지

멤버가 직접 수정할 수 있는 정보:

- 닉네임
- 프로필 이미지
- 가능 포지션
- 주발

운영자 또는 Coach가 확정하는 정보:

- 공식 라인업 포지션
- 경기 기록
- MVP
- 출석 확정 상태

## 13. 라인업 설정 화면

권한:

- Owner, Manager, Coach

구성:

- 참석 확정자
- 대기자
- 용병
- 포메이션 선택
- 작전판
- 벤치 목록
- 저장 버튼
- 참석자에게 공유 버튼

작전판 규칙:

- 선수는 드래그 또는 탭으로 배치한다.
- 각 선수는 이름, 포지션, 주발, 참석 신뢰도를 함께 볼 수 있어야 한다.
- 포메이션 저장 시 좌표와 포지션 코드를 함께 저장한다.
- 공유된 라인업은 멤버가 경기 전 확인할 수 있다.

라인업은 경기 상세 내부 탭이 아니라 `/meetings/[id]/lineup` 전체화면 작업으로 진입한다.

## 14. 전술/분석 화면

표시 정보:

- 최근 포메이션별 승률
- 포지션별 출전 기록
- 선수 조합별 결과
- 상대팀별 이전 결과
- 추천 라인업 후보
- 주의 포지션

추천 카드 구성:

- 추천 제목
- 근거 데이터
- 기대 효과
- 주의점
- 대체안

AI 추천은 초안 또는 제안으로만 표시한다. 공식 라인업과 기록은 운영자 또는 Coach가 확정한다.

## 15. 경기 기록 화면

입력 정보:

- 승/무/패
- 득점
- 실점
- 상대팀
- 사용 포메이션
- 골
- 도움
- MVP
- 선수별 실제 포지션
- 전술 변경 메모

기록 원칙:

- 결과 입력은 Owner 또는 Manager가 한다.
- 라인업/포메이션 기록은 Coach도 관리할 수 있다.
- 수정 이력은 남긴다.
- 경기 기록은 신뢰도와 승률 분석의 원천 데이터가 된다.

경기 기록은 경기 상세 내부 탭이 아니라 `/meetings/[id]/record` 전체화면 작업으로 진입한다.

## 16. 팀 허브와 작업 화면

팀 화면은 팀 상태, 멤버 수, 초대코드 상태를 요약하는 허브다.

작업 화면:

- `/team/invite`: 초대코드 복사와 재발급
- `/team/members`: 멤버 역할과 권한 관리

## 17. MY 허브

MY 화면은 프로필, 참석률, 신뢰도, 포지션 정보를 compact하게 보여준다.

작업 화면:

- `/profile/edit`: 닉네임, 프로필 이미지, 가능 포지션, 주발 편집
